# Design Pattern: Hero Scroll Indicator

## Context
Replaced standard CTA buttons in Hero sections with animated scroll indicators to create a more premium, editorial feel.

## Implementation Details
- **Component**: Custom SVG mouse icon with animated scroll wheel using Framer Motion.
- **Aesthetic**: Minimalist, slate/blue color palette, tracking-widest uppercase text.
- **Interaction**: Subtle glow and color shift on hover.
- **Animation**: Continuous loop for the scroll wheel to encourage user action.

## Purpose
Reduces visual noise and explicitly guides the user journey downwards ("défilez pour découvrir") rather than demanding an immediate click ("Suivre le projet"). Aligns with a "luxury/refined" aesthetic tone.
